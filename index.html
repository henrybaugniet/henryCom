Projects

Breeze Up Analysis in R

Whilst working for JHJW consulting I was given the opportunity to conduct my own research into how the order in which a horse is breezed can be used to predict its future performance. More explicitly, I used R studio and a number of packages within it, in order to conduct multiple regressions and some other forms of analysis. I then created a model which aims to predict the future value of a horse through variables that were recorded at horse sales. The HTML page included is generated using R markdown and relies on large database collected by JHJW consulting.
