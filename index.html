Projects

Breeze Up Analysis in R

Whilst working for JHJW consulting I was given the opportunity to conduct my own research into how the order in which a horse is breezed in combination with its breeze time at the breeze up sales can be used to predict its future performance. For someone who doesn't follow horse racing this might not mean all that much. So more explicitly I used R studio and a number of packages within it, mainly data.table and ggplot2, in order to conduct multiple regressions and some other forms of analysis. I then created a model which aims to predict the future value of a horse through recording these variables at the sales. I am currently looking to improve my model using some machine learning techniques within R.
